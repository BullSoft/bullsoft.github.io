键索引

标准Emacs快捷键序例

  • ! (Dired): 在目录模式下执行Shell。

  • " (TeX mode): 编辑TeX

  • # (Dired): 标记当前目录下所有自动保存的文件为待删除状态

  • $ (Dired): 隐藏子目录

  • % & (Dired): 标记当前目录下所有以'.bak', '.orig', '.rej'等为后缀的文件为删除状态

  • % C (Dired): 复制文件，from <RET> to <RET>

  • % d (Dired): 使用正则表达式标记文件名与之匹配的文件为待删除状态

  • % g (Dired): 使用正则表达式标记内容与之匹配的文件为选中状态

  • % H (Dired): 为指定（正则表达式）文件创建硬链接

  • % l (Dird): 把选定文件的文件名更改为小写形式

  • % m (Dired): 使用正则表达式标记文件名与之匹配的文件为选中状态

  • % R (Dired): 文件重命名

  • % S (Dired): 为指定（正则表达式）文件创建软链接

  • % u (Dired): 把选定文件的文件名更改为大写形式

  • * ! (Dired): 移除当前目录下所有文件的标记状态

  • * % (Dired): 使用正则表达式标记文件名与之匹配的文件为选中状态

  • * * (Dired): 标记当前目录下的所有可执行文件为选中状态

  • * / (Dired): 标记当前目录下的所有子目录（除了. 和 ..）为选中状态

  • * ? (Dired): 移除当前目录下标记于所有文件上的某指定状态

  • * @ (Dired): 标记当前目录下所有的软链接为选中状态。

  • * c (Dired): 更改标记于指定文件的状态（如从D → *）

  • * C-n (Dired): 移动光标到下一个有状态标记的文件上

  • * C-p (Dired)): 移动光标到上一个有状态标记的文件上

  • * DEL (Dired): 移除上一行文件的任何状态标记

  • * m (Dired): 标记当前文件为选中状态

  • * s (Dired): 标记当前目录下所有文件为选中状态

  • * t (Dired): 切换当前目录下所有文件的状态，之前是选中状态(*)的切换成未选中状态，之前是未选中状态的现在变成选中状态(*)，注意，其他状态不受影响

  • * u (Dired): 移除当前行文件的任何状态标记
